List of HTML Tags

  1. html: It defines the beginning and end of an HTML document.

  2. head: Contains meta-information about the document, such as the title and links to external resources. It doesn't display content directly on the web page.

  3. title: It helps in identifying the page.

  4. body: Contains the main content of the web page, including text, images, and other elements.

  5. h1 to h6: They provide structure to the content.

  6. p: Represents a paragraph of text.

  7. a: Creates hyperlinks to other web pages.It allows users to navigate between pages and websites.

  8. ul: Defines an unordered list.

  9. ol: Defines an ordered (numbered) list.It lists items in a specific sequential order.

  10. li: Represents a list item within ul or ol elements.It defines each item in a list.

  11. table: Creates a table to display tabular data.It organizes data into rows and columns.

  12. tr: Defines a table row within a table element.It represents a horizontal row of cells.

  13. th: Represents table header cell, it found in the first row or column.It provides labels for table data.

  14. td: Represents table data cell, containing the data in a table.It holds content such as text or other HTML elements.
